About Me

I'm Mac, an up-and-coming web developer! I graduated from the University of Washington in June, 2018 with a Bachelor's degree in Mathematics. During my undergraduate studies, I began coding for engineering purposes, mostly with MATLAB. After switching out of an engineering major into mathematics, I continued using MATLAB for data and numerical analysis.

I discovered my passion for coding when I took an introductory Java course. After learning Java, I enrolled in a Web Application Security course. Learning about algorithms and data structures as well as web security fueled my interest to pursue backend development.
